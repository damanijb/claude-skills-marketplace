# Tone & Voice Guide

Institutional voice standards for the San Bernardino County Investment Pool Report.
This is a public-facing government fiduciary stewardship document, NOT an advisory report.

## Core Principles

1. **Stewardship framing** — Present all portfolio decisions as outcomes of the Investment Policy, not active management choices.
2. **Policy authority** — Reference the Investment Policy and California Government Code §53600-53686 as the governing authority.
3. **Qualified language** — Use "may", "projected", "illustrative" instead of "will benefit" or "outperforms".
4. **Relevance filter** — Only discuss market topics directly relevant to a government fixed-income pool. Exclude equities, gold, trade policy, consumer sentiment, geopolitics.
5. **Disclaimers** — All forward-looking content must include appropriate qualifiers.

## Language Substitutions

| DO NOT say | SAY instead |
|------------|-------------|
| "positioned to benefit" | "consistent with Investment Policy objectives" |
| "yield enhancement" | "income generation within policy parameters" |
| "opportunistically" | "consistent with policy allocation limits" |
| "outperforms" | "exceeded the benchmark return" |
| "will benefit from" | "may be affected by" |
| "our strategy" | "the portfolio's structure" |
| "we positioned" | "the portfolio maintains" |
| "active management" | "portfolio management within policy guidelines" |
| "alpha generation" | (remove entirely) |
| "tactical allocation" | "allocation within established policy limits" |
| "investment thesis" | "Investment Policy objectives" |

## Section-Specific Guidance

### Executive Summary
- State facts: total market value, compliance status, key metrics
- Reference governing documents: "The Pool is governed by the Investment Policy, adopted annually by the Board of Supervisors"
- Investment objectives in priority order: Safety, Liquidity, Return

### Economic Commentary
- Focus on rates, inflation, unemployment — factors that directly affect the portfolio
- Avoid prediction: "The Federal Reserve reduced the target rate to X%" not "The Fed will continue cutting"
- Connect each observation to the portfolio: "Declining rates may affect reinvestment yields on maturing securities"

### Market Theme Slides
- Each theme should end with a sentence connecting it to the portfolio
- Use "Significance: HIGH/MODERATE/LOW" badges based on direct portfolio impact
- Example: "The yield curve has steepened to +0.74%, returning to a positively sloped shape. The shape of the yield curve affects the relative returns available across different maturities and is monitored in the context of the portfolio's maturity structure."

### Compliance Section
- State compliance status as fact: "The portfolio is in full compliance with all Investment Policy limits as of [date]"
- Show headroom: "Agency MBS: 36.3% actual vs. 40% limit — 3.7 pp headroom"
- Reference specific Gov Code sections where applicable

### Return Analysis
- Frame returns as stewardship outcomes: "The portfolio generated a total return of X%"
- Attribution: separate income return from price return
- Always note: "Income was the primary component of total return"
- Benchmark comparison: factual, no celebration of outperformance

### Horizon Analysis
- Always label as "illustrative" and "subject to assumptions"
- Note key assumptions: parallel shift, static portfolio, no reinvestment
- Connect to risk framework: "Under all five scenarios, the portfolio is projected to generate positive total return"

### Disclosures
- "This report is prepared for the Board of Supervisors and authorized participants"
- "For informational purposes only and does not constitute investment advice"
- "Market values are based on independent pricing sources (BVAL)"
- "Returns are calculated using time-weighted methodology"

## Formatting Rules

- No exclamation marks
- No first person ("we", "our") except in disclosures where "our" refers to the County
- Numbers: use commas for thousands ($16,063,518,699.58), percentage to one decimal (3.6%)
- Dates: spell out month ("January 30, 2026" not "1/30/2026")
- Abbreviations: define on first use, then abbreviate (e.g., "Agency Mortgage-Backed Securities (Agency MBS)")
